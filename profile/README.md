# Welcome to Picotainers

Picotainers is an organization dedicated to providing lightweight and efficient distroless containerized tools for the bioinformatics community. Our goal is to simplify the deployment and usage of bioinformatics software by delivering containers that are optimized for performance and ease of use.

## Repositories

### Bioinformatics Tools
- **[kallisto](https://github.com/Picotainers/kallisto)**  
  Kallisto is a software for quantifying abundances of transcripts from RNA-Seq data.

- **[minimap2](https://github.com/Picotainers/minimap2)**  
  Minimap2 is a fast sequence mapping and alignment program.

- **[samtools](https://github.com/Picotainers/samtools)**  
  Samtools is a suite of programs for interacting with high-throughput sequencing data.

- **[megahit](https://github.com/Picotainers/megahit)**  
  MEGAHIT is an ultra-fast and sensitive NGS assembler.

- **[muscle](https://github.com/Picotainers/muscle)**  
  MUSCLE performs multiple sequence alignments.

- **[fastp](https://github.com/Picotainers/fastp)**  
  Fast and comprehensive preprocessing tool for fastq data.

- **[bwa-mem2](https://github.com/Picotainers/bwa-mem2)**  
  BWA-MEM2 is the next version of the BWA-MEM.

- **[hifiasm](https://github.com/Picotainers/hifiasm)**  
  Hifiasm is a fast haplotype-resolved de novo assembler for PacBio HiFi reads.

- **[mmseqs2](https://github.com/Picotainers/mmseqs2)**  
  MMseqs2 (Million Sequence Search) is a software suite to search and cluster huge sequence sets.

- **[seqkit](https://github.com/Picotainers/seqkit)**  
  Distroless container for seqkit.

- **[bbmap](https://github.com/Picotainers/bbmap)**  
  Small BBMap container for DNA/RNA sequence alignment.
